Proyek ini merupakan hasil praktikum ke-5 mata kuliah Web Service Engineering, dengan implementasi RESTful CRUD API menggunakan Node.js dan Express.

API ini mengelola data produk sederhana (`/products`) dengan operasi:
- GET → Ambil semua produk  
- GET /:id → Ambil produk berdasarkan ID  
- POST → Tambah produk baru  
- PUT /:id → Ubah produk  
- DELETE /:id → Hapus produk
